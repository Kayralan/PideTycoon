# 🍕 Pide Tycoon (Idle Game)

Welcome to **Pide Tycoon**! This is a 2D Idle/Tycoon game developed in Unity where players start with a small traditional stone oven and grow their business to become a Pide Empire. 

Play the game on itch.io: https://kayraalan.itch.io/pidetycoon

## ✨ Features
* **Core Loop:** Bake, sell, earn, and upgrade!
* **Upgrades:** Improve your chef's baking speed and increase the income per pide.
* **Unlockable Recipes:** Progress through the game to unlock new traditional recipes (Cheese, Sucuk, Diced Meat, Mixed Pide).
* **Prestige System (Rebirth):** Reset your progress for a permanent global income multiplier.
* **UI/UX:** Custom-made rustic, cozy UI with animated progression bars and smooth transitions.

## 🛠️ Built With
* **Game Engine:** Unity 2022.x
* **Language:** C#
* **Art/UI:** Custom 2D Sprites & UI Elements
* **Packages:** TextMeshPro
